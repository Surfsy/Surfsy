Running the code: 
- Open the project in Intellij and press the green triangle button, making sure to run the "Sursfy" config

Libraries Used:
- JavaFX
- JFXoeniX - https://github.com/sshahine/JFoenix
	- JavaFX material design library
- mapjfx - https://github.com/sothawo/mapjfx
	- JavaFX MapView node
